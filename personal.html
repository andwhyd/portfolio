Personal Website

While searching for internships opportunities, I decided to create a website and portfolio after seeing a few applications request them. While my experience with web development is quite minimal, I wanted to give it a try and see if I could build one myself. This website continues to be modified and I try to keep it up to date.

Construction

The website itself is built on the Bootstrap framework, and based on a modified template found online. Using this framework saved me a lot of effort in having to write the CSS from scratch and also gave me access to a lot of prebuilt and ready to use components. With the assistance of the internet, I found the construction of this website pretty straight forward and definitely learned that a lot both HTML and CSS. As this page is a static page, it remains quite simple and easy to modify. The website is hosted using Github pages, and the link to the repository can be found below.
